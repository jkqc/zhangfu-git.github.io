<javascript高级程序设计>笔记03

笔记03是记录变量、作用域、内存问题的内容，上一章节js语句一块没有做相关的记录，主要是这些知识每个语言大致那么样。

变量

以前对变量的了解就是一个啥类型都可以存储的容器，通过本书知道了变量包含两种不同的数据类型的值。基本类型值和引用类型值
  • 基本类型值
    • 其实就是前面提到了5种基本数据类型，基本类型是按值访问
  • 引用类型值
    • js中我们是不能直接操作对象的内存空间的，我们其实都是在操作对象的引用。

在变量赋值的时候，基本类型值的变量，复制与原来是相互独立的，例如下：

var num1 = 10;
var num2 = num1;
    --num1;   //9
    num2   // 10

而引用类型复制出来是指向同一个对象，复制的变量其实是一个指针。所以当原来的值改变了，赋值的也会改变

var obj1 = new Object();
var obj2 = obj1;
obj1.name = "zhangfu";
obj2.name     //"zhangfu"  obj2的name也变了

执行环境及作用域

js中执行环境分全局与局部，全局在浏览器中指window对象，nodejs是global。每个函数都有自己的执行环境，所有局部一般都是值函数里面。在作用域方面我们要知道是没有块级作用域的，我们在if里面声明的变量是可以在if外面访问到的。

内存管理

js具有自动垃圾收集机制，现在的主流的浏览器基本都是使用标记清除垃圾收集方式。虽然我们不用操心内存管理的问题，但在语句执行数量多的时候还是有影响的。函数内容的变量和对象属性在离开执行环境的时候会自动释放，但全局的不会自动释放，所有在没有用的引用我们可以手动释放它

var obj = new Object();
obj = null;

赋值null就可以解除引用，这样就可以让它被回收。

《javascript高级程序设计》笔记02

操作符

操作符无非就是那种几种: 一元操作符, 位操作符, 布尔操作符, 乘性操作符, 加性操作符, 关系操作符, 相等操作, 条件操作符, 赋值操作符, 逗号操作符.这里记录这些操作符中应该注意的使用.

一元操作符

  • 递增和递减操作符
    var zhangfu = 1;
zhangfu ++;   //1
++ zhangfu;    //3
    从上面可以看出来后置++是操作变量之后进行+1,而前置是+1再进行变量操作
  • 一元加减操作符
    var tryzf = "bb";
+ tryzf    //NaN

在计算前都会把tryzf数值转化,当tryzf为对象的时候会调用它的valueOf()或toString()

位操作符

应该来说这个操作符一般用到的概率比较少,这里要注意一个知识点就是无符号右移(>>>),当这个数是一个负数的时候无符号右移这个值就会变的非常大,主要是负数以其绝对值的二进制进行部门形式表示

布尔操作符
布尔操作符主要有非,与,或运算,我们经常会拿它们进行如下计算:

var isSure = true;
var age = isSure && 24 || 18; //24

我们一般用或运行来给一个变量设置默认值,而用与运行进行确定.就像上面当isSure存在的时候age为24不存在的时候为18.

  • 乘性操作符
    Infinity * 0        //Nan
Infinity * (非0数值) //Infinity || -Infinity
Infinity * Infinity //Infinity至于符号问题和普通乘法一样
Infinity / Infinity // NaN
0 / 0               //NaN
(非0有限数值)  / 0    //Infinity  || -Infinity(其实会发现Infinity / 0也是一样的结果)
Infinity / (非0)    // Infinity  || -Infinity
  • 加性操作符

这个基本和数学里面的+,-差不,不过要在加运行的时候要注意与字符串的计算

"you" + 6 + 6     // 结果是"you66" 而不是you12
Infinity + (-Infinity) || -Infinity - (-Infinity)  //NaN   在这种看起来能抵消的结果都是NaN
  • 关系操作符

这个也是出频率比较高的一种运算,其要注意的一点就是在字符串做比较的时候都是根据字符编码进行比较的,在我们使用的过程中进程会遇到一些比较容易造成bug的地方.

"23" > "3"   //false
 23 > 3       //true
"23" > 3   //true
 23 > "3"   //true

从上面我们可以看到,当一个是字符,一个是数值.它就会把字符转成数值进行比较

  • 相等操作符
    这哥们应该是使用率最高的,等于和不等于会进行强制转型.这样你会发现:

    "5" == 5    //true    把字符5转成了数字5.

    所以在判断是否相等的时候我们应该尽量使用全等与不全等===与!==

  • 条件操作符
    简单的判断我们一般可以就用条件操作符比用if–else–代码量会少点.

    var max = (num1 > num2) ? num1 : num2;

    这么一个简单的取最大值的判断就完成了

  • 赋值操作符 and 逗号操作符
    赋值操作符就是从右边向左赋值,还有一些复合赋值操作,逗号操作符一般用的比较多的是多个变量的声明

    var tryzf = "zhangu",
    age = 12,
    sex = "boy";
age += 12;    // 其实就是  age = age + 12;

var say = ('hello', 'hi', 'hello hi');    //say  为 "hello hi"(可以说这种用法我在这本书上才看到)

    这就是逗号操作符和赋值操作符的一些运用.

[《javascript高级程序设计》笔记01]

一直对js没有基础的全面学习过，接触js没多久就去看jquery用来一直使用jquery。基础性的看过阮老师的JavaScript 标准参考教程。后来买了一本javascript语言精粹看了一遍，大致上对js的语言有进一步的认识。现在各种框架层出不穷，但根源都是javascript，所有学好和深入javascript显的更为重要有。最近买了《javascript高级程序设计》这本730页的javascript里的”圣经”,好好啃一遍顺便记录一些笔记。

数据类型

js有5种简单的数据类型： Undefined、Null、Boolean、Number、String;1种复杂的数据类型：object.下面简单介绍每个数据类型的特点。

  • Undefined类型（这个类型只有一个值undefined）
    1. 没有初始化的变量，它的值就是undefined
    2. 我们会用它来检测一个变量是否初始化

  • Null类型(也只有唯一的值Null)

    1. 逻辑角度Null的值是表示一个空对象指针（所以typeof null 为’object’ 注意：null == undefined为true (undefined值是派生自null值)）
    2. null作为一个对象空指标，我们一般会拿它给一个没有初始值的变量

  • Boolean类型

    1. 布尔类型有两个值true与false，注意他两是区分大小的。

    2. 可以说它就是用来判断用的，书上提到：’这true跟false与数值不是一回事，因此true不一定等1,false不一定等于0，为什么这样说呢？看下面：

       false == 0    // true (其实是把0转型了，因该是false == Boolean(0)所有结果为true)
false === 0   //false  (不是一个类型所有不等)

  • Number类型

    1. 这种类型使用IEEE754格式表示整数和浮点数
    2. number类型里主要提下0.1 + 0.2 结果不为0.3而是0.30000000000000004，主要是浮点数值最高精确是17位小数。而计算会产生舍入的误差所有就为这个值。这是使用IEEE754数值的浮点计算的通病
    3. 还有一个要提的是特殊的NaN:他是一个特殊的数值，这个数值是用于表示一个本来要返回数值操作未返回的情况。
      • 一个我不是我的也不是任何人的值 (NaN == NaN 为false)
      • 判断Nan我们可以用isNaN()函数
  • String类型
    1. 由用单或双引号包围的0～16位Unicode字符组成的字符序列
    2. 在某个值转换为字符类型的时候有一个小技巧：
      typeof("" + 1)   // String
    3. 一个值转为字符串我们可以用str.toString(这里可以传入基数)和String(str),(注意:toString()在转化null和undefined的时候会出错，这个时候就要用String())

  • Object类型

    1. 对象是一组数据和功能的集合
    2. Object类型是素有它的实例的基础，object所具有的任何方法和属性也存在更具体的对象中。

    3. Object每个实例都具有如下属性和方法

      （1）、 constructor:保存着用于创建当前对象的函数。如何理解呢！

       var obj = new Object();
obj.constructor()        //Object()

      （2）、hasOwnProperty(propertyName):用于检测给定的属性在当前对象中是否存在（而不是在实例的原型中）,
      var zhangfu = {sex: “boy”};
      zhangfu.hasOwnProperty(“sex”) //true
      Object.prototype.hasOwnProperty(“sex”) //false

      (3)、propertyIsEnumerable:用以检测给定属性是否能够使用for-in语句来枚举。

      var tryzf = [‘zhangfu’];
      tryzf.propertyIsEnumerable(0) //true
      tryzf.propertyIsEnumerable(“length”) //false

      (4)、toLocaleString: 返回字符串，该字符串与执行环境的地区对应

      var date = new Date();
date.toLocaleString();

      //同样是 7 月 23 日，在美国，toLocaleString 可能会返回 “07/23/2016 01:33:04”，而在欧洲，返回值则可能是 “23/07/2016 01:33:04”

      (5)、toString:上面提到过。

      (6)、valueOf:返回对象的字符串、数字或布尔型。简单的说本来是什么就返回什么值。

[jquery写一个简单的虚拟键盘核心代码]

最近业务需要一个虚拟键盘，本来是用jquery的Keyboard插件的，由于某些原因需要自己实现一个简单的虚拟键盘。需求的键盘很键盘就是小写字母、数字、点、删除按钮、确认按钮、取消按钮。刚开始的想想实现起来挺简单的，为了渲染更快直接把键盘写成html不用js去动态生成按钮，每个按钮写上想对应的data-id，基本排版就这样。输入iniput值不用多说就是点击获取当前点击对象的data-id让后进行字符操作赋值，然而看似简单的小键盘实现起来还是要花点心思，基本的思路就是获取目前点击的是什么，然后获取input的值，再获取input的光标位置，进行字符串的拼接最后复制回去。不过这里要注意一点input类型是number是无法获取光标的位置的，所有对number类型的input插入数据只能在最后面插入。核心代码段如下：

  • input获取光标位置

    //$node 为input的jquery对象一下雷同
function getCursorPosition($node){
  log.debug('########keypadComponent: getCursorPosition %s#####')
  var CaretPos = 0;
  var objId = $node.get(0);
  var inputType = $node.attr('type');
  if(inputType === "number"){
    CaretPos = $node.val().length;
  }else{
    if (document.selection) {  // IE Support
      objId.focus();
      var Sel = document.selection.createRange();
      Sel.moveStart('character', -objId.value.length);
      CaretPos = Sel.text.length;
    }else if(objId.selectionStart || objId.selectionStart == '0'){ // Firefox support
      CaretPos = objId.selectionStart;
    }
  }
  return CaretPos;
}

  • 添加或删除光标位置的改变

    //CaretPos获取到的光标位置，isAdd判断是添加还是删除字符
function setCursorPosition($node, CaretPos, isAdd){
  log.debug('########keypadComponent: setCursorPosition %s#####')
  var isAdd = isAdd && true || false;
  var inputType = $node.attr('type');
  if(inputType === "number"){
    return;
  }else{
    if(isAdd){
      var objId = $node.get(0);
      if(objId.setSelectionRange){
        objId.focus();
        objId.setSelectionRange(CaretPos + 1, CaretPos + 1);
      }else if (objId.createTextRange) {
        var range = objId.createTextRange();
        range.collapse(true);
        range.moveEnd('character', CaretPos + 1);
        range.moveStart('character', CaretPos + 1);
        range.select();
      }
    }else{
      var objId = $node.get(0);
      if(objId.setSelectionRange){
        objId.focus();
        objId.setSelectionRange(CaretPos - 1, CaretPos - 1);
      }else if (objId.createTextRange) {
        var range = objId.createTextRange();
        range.collapse(true);
        range.moveEnd('character', CaretPos - 1);
        range.moveStart('character', CaretPos - 1);
        range.select();
      }
    }
  }
}

  • 文字的插入

    //$node 是input的jquery对象 keyStr是虚拟键盘获取到的data-id
function inserTexts($node, keyStr){
  log.debug('########keypadComponent: inserText %s#####');
    CaretPos = getCursorPosition($node),
    inputStr = $node.val(),
    objId = $node.get(0),
    inputType = $node.attr('type');
  log.debug(inputType)
  if(inputType !== "number"){
    if(CaretPos !== inputStr.length - 1){
      var startText = inputStr.substring(0, CaretPos);
      var endText = inputStr.substring(objId.selectionEnd);
      var newStr = startText + keyStr + endText;
      $node.val(newStr);
    }else{
      $node.val(inputStr + keyStr);
    }
  }else{
    $node.val(Number(inputStr + keyStr));
  }
  setCursorPosition($node, CaretPos, true);

}
  • 删除字符
    //currInputValue 为当前input的value值
function removeTexts($node, currInputValue){
  log.debug('########keypadComponent: removeText %s#####')
  var CaretPos = getCursorPosition($node);
  $node.val(currInputValue && currInputValue.substring(0, CaretPos - 1 ) + currInputValue.substring(CaretPos, currInputValue.length));
  setCursorPosition($node, CaretPos, false);
}
    以上是整个键盘比较核心的代码，主要的注意点就是input类型为数值型的时候，主要的陌生点就是光标的获取和设置
这里要提到键盘的一个体验性关闭功能的实现，当时有这样一个需求就是点击键盘外的区域隐藏键盘，但点击input不隐藏键盘。刚开始想到的是给键盘外加一个透明的全屏层把点击事件绑定给它，但发现这样没办法实现点击input直接获取光标而且键盘还在的需求。然后又想到拿input的focus事件的注意让它获取焦点显示，让它失去焦点的时候隐藏键盘，但发现不可行点击虚拟键盘就没了。
最后通过一个isFocus标识来实现需要的效果：
$(document).unbind('click').bind('click', function(e){

  if(!isFocus){
    关闭键盘
  }
});
监听document的点击事件，判断是否关闭键盘(在点击input和虚拟键盘的时候给isFocus赋值true).这样就实现了点击键盘和input以外的区域实现关闭

以上就是实现一个简单键盘思路和注意点。

[工作中经常使用的几个Git命令]

现在的项目开发基本都是用Git管理项目版本，配合github远程仓库来协同工作和管理项目。今天把这个月使用频率最高的几条命令做一个记录：

  • git checkout branch_name //切换到某一个分子
  • git checkout 文件路径 //还原某个文件
  • git add file_name // 文件修改添加到暂存区
  • git commit -m “XXXXX” // 把暂存区的所有内容提交到当前分支,-m ‘xxx’进行描述
  • git push //提交到远程分支
  • git pull //从远程仓库把最新更新拉回来
  • git stash //暂存目前工作上的分支
  • git stash pop //恢复前面暂存的分支
  • git branch -m old branch new branch //重名分支
  • git checkout -b new branch name // 创建分支并切换到改分支(创建的分支文件和当前所在分支的文件一样)
以上就是这一个月来经常用到一些基本git命令
