[工作中经常使用的几个Git命令]

发表于 2016-07-17   |  

现在的项目开发基本都是用Git管理项目版本，配合github远程仓库来协同工作和管理项目。今天把这个月使用频率最高的几条命令做一个记录：

  • git checkout branch_name //切换到某一个分子
  • git checkout 文件路径 //还原某个文件
  • git add file_name // 文件修改添加到暂存区
  • git commit -m “XXXXX” // 把暂存区的所有内容提交到当前分支,-m ‘xxx’进行描述
  • git push //提交到远程分支
  • git pull //从远程仓库把最新更新拉回来
  • git stash //暂存目前工作上的分支
  • git stash pop //恢复前面暂存的分支
  • git branch -m old branch new branch //重名分支
  • git checkout -b new branch name // 创建分支并切换到改分支(创建的分支文件和当前所在分支的文件一样)
以上就是这一个月来经常用到一些基本git命令

[前端小白起源说]

发表于 2016-07-11   |  

13年入学zjy，很多同学一样进入大学后开始迷途。体验了两年多的大学生活突然意识毕业的危机感，想想自己读了这么多年计算机专业毕业要是找不到一份跟专业有关的工作那岂不是很浪费青春。偶然机会接触html，刚开始啥都不会，去图书馆借了一本html+css的入门书看看打打，通过网上的视频教材学习js，渐渐的对写页面有初步的认识。后来接触了jquery发现有了它世界有美好了许多，记得用jqury写的第一个功能是我同学网站的一个菜单。鼠标移入显示，移出隐藏。那时候拿了一本厚厚jquery入门数据看了一遍，说其在的到最后基本没有记录几个。
2015-8和同学一起开发一个电商型网站，也是那次机会让我学习了很多东西。对ajax请求DOM的操作有了一个初步的认识。
2015-12进入一家外包公司实习，可能和小部分同学一样，我日常工作就是切图切图切图，这些工作总的来说是在锻炼布局和样式，对js上的运用比较少，可能最多就用几个通过的插件，幸运的话可以写几百行的js代码，各种局限最后让我选择了离开贵地。
2016-6来到现家，为期35天的工作让我重新认识了前端，也让我从切图仔慢慢的走向前端开发的道路。node、npm、webpack、bower、less、jade、组件化这也许才是我想要的前端开发。以前jquery用到最多的只是选择器、动画、简单的一些事件。如今$.data()、trigger()、$.when()、done()、fail()、promise()等又让我玩起新高度，前端的路还很长很坎坷，要学习的东西还有很多。不忘初心——–
