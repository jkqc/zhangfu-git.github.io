Array类型

创建数据的方式有多种，我可以用var myArray = new Array()、也可以直接var myArray= [],javascript中的数组很特别，我们可以更改它的length，来移除最后一项例如：

var myArray = ['sex', 'name', 'age'];
myArray.length = 2;
myArray[2]        //undefined

对数组的操作有以下方法：

  • push()——————-//可以接收任意数量的参数，插入数组尾部
  • pop()———–// 从数字尾部移除一项，返回删除项
  • shift()———-// 移除第一项，并返回改项
  • unshift()———//在数组前端添加任意个，并返回新的数组
  • reverse()———//颠倒原来的数据顺序

  • sort()————//按字符串排序

    sort()还可以传入一个比较函数，我可以通过比较函数返回的值进行数字的排序，比较函数的第一个参数小于第二个参数返回负数，两个参数相等返回0,第一个参数大于第二个参数返回一个正数。

  • concat()—-//可以传入一个或多个数组，把这些数组加入原数组尾部

  • slice()—//接收两参数，该方法返回起始和结束位置之间的项
    注意：当传入的为负数，则参数用length作为参数。如果结束小于开始位置，则返回空数组

  • splice()—//这哥们强大，可删除、可替换、可插入,返回的都是删除的项。

    var  colors = ['red', 'green', 'blue'];
colors.splice(0, 1);   //删除了第一项0表示其实位置，1表示要删除的个数：['green','blue']
colors.splice(1, 0, 'yellow')  //从位置1开始删除0个插入yellow :['green', 'yellow', 'blue']
colors.splice(1, 1, 'gray')  //从1开始删除1个插入yellow------:['green', 'gray', 'blue']
  • indexOf()—–//接收两个参数，第一个要找的项第二个开始找的起点.这哥们是从数组的开头开始向后找
  • lastIndexOf() —-//这哥们和上面那个哥们方向相反，功能一样。当他们没找到都是返回-1,要注意的是第一个参数和对比的每个项是使用全等操作符
    迭代方法
  • every()
  • filter()
  • forEach
  • map()

  • some()

    他们接收两个参数，一个函数、一个作用域对象。下面用代码说明他们各自的用处

    var numbers = [1,2,3,4,5];
var everyResult = numbers.every(function(item, index, array){
    return (item > 2)
});
console.log(everyResult)   //false 要数组每项都满足才返回true

var someResult = numbers.some(function(item, index, array){
    return (item > 2)
});
console.log(someResult)    //true  有一个项满足条件就返回true

var filterResult = numbers.filter(function(item, index, array){
  return (itme > 2)
});
console.log(filterResult)  //[3,4,5]  过滤出大于2的项并返回新数组

var mapResult = numbers.map(function(item, index, array){
  return item * 2;
});
console.log(mapResult)  //[2,4,6,8,10] 把每进行操作，然后返回操作后的数组

forEach()  //这个方法没有返回值，一般我用它来遍历
  • reduce()—//迭代数组的所有项，然后构建一个最终返回值.方向从left开始

  • reduceRight() –//和上面一样，方向从right开始.看代码好理解：

    var values = [1,2,3,4,5];
    var sum = values.reduce(function(prev, cur, index, array){

    return prev + cur;

    });
    sum // 15;
